Bienvenido a las Actualizaciones

Esta página ofrece las siguientes funcionalidades:

Evaluación de Salud

La página permite realizar un análisis completo de salud mediante cuestionarios interactivos.

Ejercicios de Memoria

Incluye ejercicios para mejorar la memoria, diseñados para estimular la mente.

Recordatorios

Recibe notificaciones personalizadas para asegurarte de no olvidar ninguna tarea importante.

Panel del Cuidador

El cuidador tiene acceso a un panel para monitorear el progreso y bienestar del usuario.
